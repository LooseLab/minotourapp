    Sequencing Speed

    Calculating Sequencing Speed - this can take time and will be at least ten minutes into the run. So - Don't Panic.
    Speed of DNA moving through the pore is estimated from fastq data alone. More accurate calculations can be made from the raw Nanopore data.
    This feature is provided to allow users to estimate if speed is falling outside reasonable parameters.
    Speed estimates will not be correct for runs with extremely long read lengths or those where input library is limiting.
